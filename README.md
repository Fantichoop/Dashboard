# Dashboard
Dashboard — это информационная панель, которая получает данные из других систем и отображает их в понятном виде. На дашбордах используют текст, графики, диаграммы и другие средства визуализации.
## Цель
Суть проекта в том, что мы разрабатываем визуализацию данных с IoT устройств в реальном времени, в частности, это будет анализ данных о погоде в Томске в разный период времени.
## Инсрументы
Мы использовали такие инструменты, как:
- Flask
- git
- github
- Python
- UML
- SQLite
- SQLAlchemy
- pandas
- OWM
